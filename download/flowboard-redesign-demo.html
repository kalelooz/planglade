Design system summary
Typography

Satoshi, 13-14px body, 12px metadata, 18-20px titles. Compact line heights and no oversized display text inside product surfaces.

Color

Soft off-white, light gray sidebars, crisp dividers, dark text, muted labels, one calm teal accent.

Structure

Rows, dividers, tabs, sidebars, drawers, compact toolbars. Cards only when an object must be physically moved, such as Kanban cards.

Spacing

4/8px rhythm, 48px topbar, 32px nav rows, 46px work item rows, 24px content gutters on desktop.

Radius and borders

7-8px controls, 12px overlays, crisp 1px borders, low shadow only for floating overlays and drawers.

Interaction states

Selected row uses a teal left rail. Hover states use a faint gray surface. Command palette and drawer are overlay patterns.
