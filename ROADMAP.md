# Roadmap

PlanGlade is a calm, light-first, open-source project workspace for solo users and small teams.

This roadmap is directional. It avoids dates on purpose. Items may move as the product and self-hosting path mature.

## Available Today

- Home command center.
- Inbox capture and triage.
- Tasks with list and board views.
- Projects and Project Home.
- Notes and project context.
- Calendar over task due dates.
- Settings.
- Workspace JSON export and guarded import.
- Early local/developer self-host docs.

## Next

- Timeline polish.
- Task dependencies.
- Recurring tasks.
- Stronger self-host path.
- Docker support after it is implemented and tested.
- Security hardening for auth, workspace boundaries, import/export, attachments, and deployment guidance.

## Later

- Collaboration.
- Hosted cloud.
- Billing.
- Admin/team features.
- Time tracking and reporting.
- Work Map / dependency visualization.
- AI assistance only if it becomes useful after the core app is trustworthy.

## Rules For Public Claims

- Do not market later items as available.
- Do not claim production-ready self-hosting yet.
- Do not claim pricing, hosted cloud, AI assistance, Docker, or enterprise/admin features are available today.
- Keep MVP claims tied to real working product surfaces.
